Uso de la API

Todos los datos son ficheros JSON estáticos servidos desde GitHub Pages. Sin autenticación, sin rate limiting, con CORS habilitado.

// Registro de autores
GET /data/authors.json

// Datos de un autor por era
GET /data/martinez-ares/chunk-1980-1999.json
GET /data/martinez-ares/chunk-2000-2019.json

// Índice de búsqueda global
GET /data/search-index.json
// Ejemplo en JavaScript
const BASE = 'https://nandovejer.github.io/ares-api/data/';

const authors = await fetch(BASE + 'authors.json').then(r => r.json());

const works = await fetch(BASE + 'martinez-ares/chunk-1980-1999.json')
  .then(r => r.json());
